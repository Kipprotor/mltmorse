このプログラム/モジュールではモールス信号の対応表をyaml形式で以下のように設定することができます。

```yaml:numberAndSymbol.yaml
number:
  1: ".----"
  2: "..---"
  3: "...--"
  4: "....-"
  5: "....."
  6: "-...."
  7: "--..."
  8: "---.."
  9: "----."
  0: "-----"

symbol:
  "+": ".-.-."
  "-": "-....-"
  #...
```

```yaml
alphabet:
  a: ".-"
  b: "-..."
  #...
```
```yaml
japnanese:
  あ: "--.--"
  い: ".-"
  う: "..-"
  #...
```
